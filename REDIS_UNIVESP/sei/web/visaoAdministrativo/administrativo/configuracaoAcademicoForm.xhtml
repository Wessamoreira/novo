Legenda:
A = Ano
S = Semestre
U = Unidade de Ensino
C = Curso
B = Abreviatura do Curso
I = Incremento
Pode ser usado ponto (.) vírgula (,) barras (/)
ou qualquer tipo de caracter para compor a máscara. Exemplo:(AAAA+SS+IIII)/4 Ao habilitar essa opção será incluído na matrícula/renovação as disciplinas de dependência primeiro e caso tenha sobrado carga horária seram incluídas as disciplinas regulares.
